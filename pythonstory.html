Problems I Encountered Durring the Process

Some of the dificulties I faced when creating this project was that my state machine wouldn't work. For example I had too many while statements when I only needed one. Whenever I tried to type in the letter for it to go to the next part of the story it wouldn't work. Another problem that I ran into while working on this project was that I messed up half of my story and had to re-name all of my functions. Another problem I ran into was that when I wanted to go to the next option it would skip directly to the end. I fixed that problem by creating a end statement on my final function. I also had the problem of not giving my function enough time to be displayed so I added a 5 second timer between the different functions so they wouldnt be skipped.
